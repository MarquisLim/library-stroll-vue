# Задание 5. Комплексное проектирование с применением UML и шаблонов GRASP

## Описание задания

**Цель:** Объединить все этапы анализа, моделирования и проектирования в полном цикле с применением шаблонов проектирования.

**Предметная область:** Library Stroll - Платформа для художников

## Структура проекта

```
assignment-5/
├── README.md                    # Этот файл
├── REPORT.md                    # Полный отчет
├── uml-diagrams/               # UML диаграммы
│   ├── use-case/              # Use Case диаграммы
│   ├── activity/              # Activity диаграммы
│   ├── sequence/              # Sequence диаграммы
│   ├── class/                 # Class диаграммы
│   ├── component/             # Component диаграммы
│   └── deployment/            # Deployment диаграммы
├── grasp-patterns/             # GRASP шаблоны
│   ├── patterns.md            # Описание примененных шаблонов
│   └── implementation/        # Реализация шаблонов
├── architecture/               # Архитектура приложения
│   ├── design.md              # Описание архитектуры
│   └── code/                  # Код реализации
└── evaluation/                # Оценка проектных решений
    ├── stability.md           # Устойчивость к изменениям
    ├── reusability.md         # Повторное использование
    ├── solid.md               # Соответствие SOLID
    └── consistency.md         # Согласованность моделей
```

## Требования

1. ✅ Построить набор UML-диаграмм
2. ✅ Определить роли и ответственности классов по GRASP
3. ✅ Создать архитектуру с реализацией GRASP-шаблонов
4. ✅ Выполнить оценку проектных решений
5. ✅ Подготовить отчёт
