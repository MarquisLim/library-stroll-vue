# Задание 3. Разработка интегрированной системы структурного и объектного моделирования

## Описание задания

**Цель:** Объединить преимущества SADT/DFD и UML-подходов в одной системе.

**Предметная область:** Library Stroll - Платформа для художников

## Структура проекта

```
assignment-3/
├── README.md                    # Главный файл с описанием
├── REPORT.md                    # Отчет по заданию
├── sadt/                        # SADT модели
│   ├── a0-diagram.md           # Диаграмма A0 (верхний уровень)
│   └── a1-diagrams.md          # Диаграммы A1, A2, A3...
├── dfd/                         # DFD диаграммы
│   ├── level-0.md              # DFD уровня 0 (контекстная диаграмма)
│   ├── level-1.md              # DFD уровня 1
│   └── level-2.md              # DFD уровня 2 (элементарные процессы)
├── uml/                         # UML модели
│   ├── class-diagram.md        # Диаграмма классов
│   ├── sequence-diagrams.md    # Диаграммы последовательностей
│   └── activity-diagrams.md    # Диаграммы активностей
├── translation/                 # Модель трансляции DFD → UML
│   ├── mapping-table.md        # Таблица сопоставления элементов
│   └── translation-rules.md    # Правила трансляции
├── generator/                   # Автоматический генератор
│   ├── dfd-to-uml.py           # Python скрипт генерации
│   ├── README.md               # Инструкции по использованию
│   └── examples/              # Примеры генерации
└── analysis/                    # Анализ трансляции
    ├── principles-analysis.md  # Анализ принципов
    └── comparison.md          # Сравнение до и после

```

## Требования

1. Построить SADT-модель верхнего уровня (A0 → A1…)
2. Разработать DFD до уровня элементарных процессов
3. Создать объектную модель UML (классы, последовательности, активности)
4. Создать модель трансляции DFD → UML
5. Реализовать автоматический генератор UML из DFD
6. Проанализировать принципы трансляции
