# DFD Уровень 0 - Контекстная диаграмма

## Описание

Контекстная диаграмма показывает систему Library Stroll как единый процесс с внешними сущностями.

## Диаграмма (Mermaid)

```mermaid
flowchart LR
    Users[Пользователи] -->|Запросы, данные| System[Система Library Stroll]
    System -->|Контент, уведомления| Users
    
    Admin[Администраторы] -->|Управление, модерация| System
    System -->|Отчеты, статистика| Admin
    
    External[Внешние сервисы<br/>Pusher, CDN] -->|Real-time, файлы| System
    System -->|Запросы| External
    
    style System fill:#e1f5ff
    style Users fill:#fff3e0
    style Admin fill:#f3e5f5
    style External fill:#e8f5e9
```

## Внешние сущности

1. **Пользователи** - художники и посетители платформы
2. **Администраторы** - модераторы и администраторы системы
3. **Внешние сервисы** - Pusher (real-time), CDN (файлы)

## Потоки данных

### От Пользователей к Системе:
- Запросы на регистрацию/вход
- Художественные работы (загрузка)
- Комментарии и сообщения
- Лайки и подписки

### От Системы к Пользователям:
- Отображение контента
- Уведомления
- Результаты операций

### От Администраторов к Системе:
- Команды модерации
- Настройки системы
- Управление пользователями

### От Системы к Администраторам:
- Отчеты о модерации
- Статистика системы
- Логи активности
