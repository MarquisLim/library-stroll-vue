# SADT Диаграммы A1 - Декомпозиция A0

## Описание

Диаграммы A1 представляют декомпозицию верхнего уровня на основные функциональные области.

## Диаграмма A1: Управление пользователями и аутентификацией

```mermaid
flowchart LR
    subgraph A1["A1: Управление пользователями"]
        direction TB
        I1[Вход:<br/>Данные регистрации/входа]
        P1[Процесс:<br/>Аутентификация,<br/>управление профилями]
        O1[Выход:<br/>Сессии пользователей,<br/>профили]
        C1[Управление:<br/>Политики безопасности]
        M1[Механизм:<br/>Laravel Sanctum,<br/>User Model]
        
        I1 --> P1
        C1 --> P1
        P1 --> O1
        M1 --> P1
    end
    
    style A1 fill:#e1f5ff
```
